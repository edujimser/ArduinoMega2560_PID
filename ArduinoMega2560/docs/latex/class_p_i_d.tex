\doxysubsection{Descripción detallada}
Controlador \doxylink{class_p_i_d}{PID} con anti-\/windup, derivative-\/on-\/measurement y limitación de cambio de salida (slew-\/rate limit). 

Este controlador está diseñado para sistemas sensibles como motores paso a paso, actuadores lentos o sistemas donde se requiere suavidad y estabilidad.
