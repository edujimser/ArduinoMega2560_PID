◆ runTest()

bool DiagnosticsEEPROM::runTest ( int address = 0)
static

Runs a diagnostic test on the EEPROM at the specified address. Writes a test value, reads it back, and verifies integrity.

Ejecuta un test de diagnóstico sobre la EEPROM en la dirección indicada.

Parámetros
addressEEPROM address to test (default is 0).
Devuelve
true if EEPROM responds correctly, false otherwise.

El procedimiento realiza:

  • Escritura de un valor de prueba en la EEPROM.
  • Lectura del valor almacenado.
  • Comparación entre valor escrito y leído.
  • Informe del estado de memoria libre y usada.
  • Limpieza de la dirección probada.
Parámetros
addressDirección de EEPROM a probar (por defecto 0).
Devuelve
true si la EEPROM responde correctamente, false en caso contrario.
