◆ diagnoseUARTStream()

void diagnoseUARTStream ( HardwareSerial & serialPort,
const char * portName )

Realiza un diagnóstico sobre un puerto UART específico.

El procedimiento:

  • Inicia el puerto UART a 9600 baudios.
  • Espera durante 3 segundos la llegada de datos.
  • Si recibe datos, los imprime por consola.
  • Si no recibe nada, informa que no hubo actividad.
Parámetros
serialPortReferencia al objeto HardwareSerial (por ejemplo, Serial1 o Serial2).
portNameNombre del puerto UART para mostrar en consola.
