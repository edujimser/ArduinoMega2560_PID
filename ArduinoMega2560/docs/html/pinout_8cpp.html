void diagnoseAnalog ( )

Diagnostica el voltaje presente en los pines analógicos del sistema.

Diagnóstico de pines analógicos.

Recorre todos los pines definidos en Pins::ANALOG, los configura como entrada, estabiliza la señal y realiza una lectura analógica. Convierte el valor leído a voltaje (asumiendo referencia de 5V) y lo muestra por Serial.

Nota
Incluye pequeñas demoras para estabilizar la señal.
Atención
Asegúrate de que el sistema use referencia de 5V para lecturas precisas.

void diagnoseGPIO ( )

Diagnóstico de pines GPIO configurados como entrada.

Diagnóstico de pines GPIO.

Configura cada pin GPIO como INPUT, estabiliza la señal y lee su estado. Informa si el pin tiene voltaje externo (HIGH) o está en LOW/GND.

Nota
Útil para detectar si un pin está conectado a tierra o flotando.

void diagnosePWM ( )

Detecta la presencia de voltaje externo en pines PWM.

Diagnóstico de pines PWM.

Configura cada pin PWM como entrada, estabiliza la señal y lee su estado digital. Informa si hay voltaje (HIGH) o no (LOW/GND).

Nota
No mide voltaje real, solo estado lógico.
Para medir voltaje real, usar pines analógicos.

void fullDiagnosticsPins ( )

Ejecuta un diagnóstico completo de los pines digitales GPIO, PWM y analógicos.

Ejecuta todos los diagnósticos de pines.

Esta función:

  • Muestra un mensaje de inicio del diagnóstico.
  • Llama a diagnoseAnalog(), diagnoseGPIO() y diagnosePWM().

Útil como prueba inicial del hardware al arrancar el sistema.
