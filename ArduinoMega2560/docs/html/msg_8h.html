Prints an enriched error message to the serial monitor, including contextual information.

This function generates a structured output that helps identify the origin of runtime errors. The output includes:

  • Compilation date and time
  • Source file name
  • Function name from which it is invoked
  • Line number where the function was called
  • Custom error message

Output format: [date time] file::function (Line N) ❌ ERROR ➤ message

Example output: [Aug 7 2025 01:23:00] sensor.cpp::readSensor (Line 42) ❌ ERROR ➤ Failed to initialize sensor

Parámetros
messageDescription of the specific error
fileSource file name (FILE)
functionFunction name where the error was detected (FUNCTION)
dateCompilation date (DATE)
timeCompilation time (TIME)
lineLine number in the file where the function is invoked (LINE)

This function builds a single line consisting of a combined text that includes the message, file, function, date, and time, surrounded by decorative characters on the left and right to achieve a total line width specified by the user.
