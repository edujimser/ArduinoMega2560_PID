\doxysubsection{Descripción detallada}
Controlador para motores paso a paso 28BYJ-\/48 con driver ULN2003. 

Esta clase permite:\+
\begin{DoxyItemize}
\item Controlar pasos individuales (adelante/\+atrás)
\item Establecer velocidad en RPM
\item Realizar movimientos relativos en grados
\item Obtener información del progreso del movimiento
\item Depurar el comportamiento mediante mensajes por consola
\end{DoxyItemize}

La clase está organizada por familias funcionales para facilitar la lectura, mantenimiento y generación de documentación.
