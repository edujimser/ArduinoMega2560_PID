\doxysubsection{Descripción detallada}
Representa un pin del sistema con nombre simbólico, número físico y familia funcional. 

Esta estructura permite describir cada pin de forma clara y tipada, facilitando diagnósticos, documentación y uso en tiempo de compilación.

\hline
