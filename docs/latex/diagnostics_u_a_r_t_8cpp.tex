void diagnose\+UARTStream (\begin{DoxyParamCaption}\item[{Hardware\+Serial \&}]{serial\+Port}{, }\item[{const char \texorpdfstring{$\ast$}{*}}]{port\+Name}{}\end{DoxyParamCaption})}



Realiza un diagnóstico sobre un puerto UART específico. 

El procedimiento:\+
\begin{DoxyItemize}
\item Inicia el puerto UART a 9600 baudios.
\item Espera durante 3 segundos la llegada de datos.
\item Si recibe datos, los imprime por consola.
\item Si no recibe nada, informa que no hubo actividad.
\end{DoxyItemize}


\begin{DoxyParams}{Parámetros}
{\em serial\+Port} & Referencia al objeto Hardware\+Serial (por ejemplo, Serial1 o Serial2). \\
\hline
{\em port\+Name} & Nombre del puerto UART para mostrar en consola. \\
\hline
\end{DoxyParams}
