\doxysubsection{Descripción detallada}
Archivo principal de cabecera del firmware. Contiene metadatos del firmware, includes del sistema, configuración de pines, diagnósticos, motor paso a paso y controlador \doxylink{class_p_i_d}{PID}. 

Este archivo centraliza todas las dependencias necesarias para el funcionamiento del firmware en Arduino Mega 2560.
