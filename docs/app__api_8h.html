◆ dboot_get_version()

uint8_t dboot_get_version ( uint16_t * ver)

Read the version of the bootloader from bootloader memory. This is the version of the bootloader, not the version of the API. The bootloader verison is (AVRDBG_BOOT_VERSION_MAJOR << 8) | (AVRDBG_BOOT_VERSION_MINOR) as defined in bootapi.c

Parámetros
ver(out) - the version of the bootloader
Devuelve
BOOT_OK on success.
