  • Bit 0: debugMode

  • Bit 2: diagnoseAnalog
  • Bit 3: diagnoseGPIO
  • Bit 4: diagnosePWM
  • Bit 5: diagnoseUART
  • Bit 6: diagnoseEEPROM
  • Bit 7: version
